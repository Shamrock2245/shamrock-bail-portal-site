# Wix AI Theme Assistant - Key Findings

## What the Theme Assistant Can Do

Based on Wix documentation:

1. **Conversational Chat Interface** - Users interact through chat to describe needs
2. **Color Palette Selection** - Selecting and generating color schemes
3. **Font Selection** - Choosing typography that matches brand
4. **Extract Colors from Logo** - Can pull colors from uploaded logo
5. **Experiment with Themes** - Try different theme variations
6. **Generate New Themes** - Create entirely new theme based on description
7. **Shuffle Colors** - Switch primary/secondary colors
8. **Layout Changes** - Modify page layouts
9. **Real-time Suggestions** - Provides guidance while working

## Key Documentation Points

From the press release (Sep 12, 2024):
> "Users interact with the theme assistant through a conversational chat and can ask the theme assistant for guidance about the design process, from selecting color palettes and fonts to extracting colors from their logo and experimenting with various themes."

> "The assistant can recommend features, direct users to specific areas in the Wix dashboard to make relevant changes, and provide advice on optimizing fonts, colors, and other theme elements."

## Important Limitation

From the FAQ:
> "It is not possible to use the AI Website Builder to edit existing sites, you can only use it to create new sites. However, there are numerous AI tools you can use in the Editor to edit your site. For example, you can generate whole new page sections or redesign your site theme."

## What to Include in Theme Prompts

Based on documentation, the AI responds to:
- **Business type and description**
- **Design preferences** (modern, classic, bold, minimal)
- **Color preferences** (specific hex codes or color names)
- **Font preferences** (font family names)
- **Industry context** (helps AI understand appropriate aesthetics)
- **Emotional tone** (professional, friendly, urgent, calming)

## Theme Generation Options

When editing the site brief, you can specify:
- **Site profile/description** - Business details
- **Site structure** - Sections and pages to include
- **Theme** - Font and color changes
- **Layout** - Design structure changes
